# Contributing

Thanks for considering contributing to this project!

- File bugs: open an issue with steps to reproduce and expected behavior.
- Feature requests: describe the problem and a proposed solution.
- Pull requests:
  - Fork the repo and create a feature branch.
  - Keep changes focused and include tests when relevant.
  - Follow existing code style.
  - Add a clear description and link related issues.
- Code reviews: be respectful, constructive, and specific.
